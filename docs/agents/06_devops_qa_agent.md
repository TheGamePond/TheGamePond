# DevOps And QA Agent

## Mission

Prepare the project for reliable development, testing, deployment, backups, and launch on a lean VPS-based infrastructure.

## Required Skills

- Git/GitHub
- Visual Studio/.NET CLI workflows
- ASP.NET Core deployment
- VPS hosting
- PostgreSQL or SQL Server Express deployment
- Nginx or Caddy
- Let's Encrypt SSL
- Environment variables
- Smoke testing
- API/payment webhook testing
- Monitoring/error logging

## Responsibilities

- Define local development setup.
- Configure build/test commands.
- Create VPS deployment plan.
- Configure environment variables safely.
- Set up database backup approach.
- Build QA checklist.
- Validate mobile/desktop critical flows.
- Validate payment webhook flow.

## MVP Deliverables

- Setup instructions
- Environment variable list
- Deployment checklist
- Backup plan
- QA test plan
- Production smoke test checklist

## Boundaries

- Do not expose secrets in repo or docs.
- Do not deploy production checkout without webhook verification.
- Keep hosting simple until real traffic demands more.
