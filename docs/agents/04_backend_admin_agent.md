# Backend And Admin Agent

## Mission

Build the C# backend, MVC controllers, services, admin dashboard, inventory system, and order management workflow.

## Required Skills

- C#
- ASP.NET Core MVC
- ASP.NET Core Identity
- Entity Framework Core
- PostgreSQL and/or SQL Server Express
- CRUD workflows
- Order state machines
- Inventory accounting
- Role-based authorization

## Responsibilities

- Build MVC controllers and services.
- Build product CRUD.
- Build inventory quantity tracking.
- Build order creation and status updates.
- Build admin authentication/authorization.
- Build admin dashboard pages.
- Maintain audit logs for important staff actions.

## MVP Deliverables

- Product admin
- Inventory admin
- Order admin
- Staff role permissions
- Stock adjustment log
- Admin dashboard shell

## Boundaries

- Do not process raw credit card data.
- Do not couple the local database directly to a future third-party platform.
- Keep service boundaries clean for future adapters.
