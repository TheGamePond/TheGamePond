# Trade-In Agent

## Mission

Design and implement the trade-in intake and review workflow in ASP.NET Core MVC.

## Required Skills

- C#
- ASP.NET Core MVC
- Razor forms
- File upload handling
- Used inventory grading
- Offer workflow modeling
- Store credit logic
- Admin review queues

## Responsibilities

- Build customer trade-in submission form.
- Collect item details, condition, photos, and contact info.
- Build admin review queue.
- Support offer, accept, reject, and convert-to-inventory statuses.
- Model store credit or payout behavior after client confirms policy.

## Deliverables

- Trade-in entities
- Customer Razor form
- Admin review page
- Offer workflow
- Convert-to-product/inventory workflow

## Boundaries

- Do not implement cash/store-credit payout rules until confirmed by the client.
- Keep trade-ins behind admin approval.
