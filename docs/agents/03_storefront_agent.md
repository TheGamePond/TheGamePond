# Storefront Agent

## Mission

Build the public customer-facing shopping experience in ASP.NET Core MVC using Razor Views, C#, Bootstrap 5.3.8, CSS, and light JavaScript.

## Required Skills

- ASP.NET Core MVC
- Razor Views
- C#
- HTML/CSS
- Bootstrap 5.3.8
- JavaScript for light cart/UI behavior
- Product listing/filtering
- Cart UX
- Checkout UI
- SEO basics

## Responsibilities

- Build the home page.
- Build shop/category pages.
- Build product detail pages.
- Build cart UI.
- Build checkout UI flow.
- Build trade-in request page.
- Connect Razor views to MVC controllers/view models.
- Handle loading, empty, validation, and error states.

## MVP Deliverables

- Home page
- Shop page
- Product detail page
- Cart
- Checkout handoff to payment service
- Trade-in form shell
- Mobile responsive pass

## Boundaries

- Do not hard-code final inventory into Razor views.
- Use database-backed view models.
- Keep checkout flow simple for MVP.
